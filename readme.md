**–ö–æ–ø–∏—Ä–æ–≤–∞–Ω–∏–µ –∏ –ø–µ—Ä–µ–º–µ—â–µ–Ω–∏–µ**

- `cat file.txt` (–æ—Ç –∞–Ω–≥–ª. *con**cat**enate and print*, ¬´–æ–±—ä–µ–¥–∏–Ω–∏—Ç—å –∏ —Ä–∞—Å–ø–µ—á–∞—Ç–∞—Ç—å¬ª) ‚Äî —Ä–∞—Å–ø–µ—á–∞—Ç–∞–π —Å–æ–¥–µ—Ä–∂–∏–º–æ–µ —Ç–µ–∫—Å—Ç–æ–≤–æ–≥–æ —Ñ–∞–π–ª–∞ `file.txt`.

**–£–¥–∞–ª–µ–Ω–∏–µ**

### –ü–æ–ª–µ–∑–Ω—ã–µ –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç–∏

### –ß—Ç–æ —Ç–∞–∫–æ–µ Git?


**Git** - —ç—Ç–æ —Å–∏—Å—Ç–µ–º–∞ –∫–æ–Ω—Ç—Ä–æ–ª—è –≤–µ—Ä—Å–∏–π, –∫–æ—Ç–æ—Ä–∞—è –ø–æ–º–æ–≥–∞–µ—Ç –æ—Ç—Å–ª–µ–∂–∏–≤–∞—Ç—å –∏–∑–º–µ–Ω–µ–Ω–∏—è –≤ –ø—Ä–æ–µ–∫—Ç–µ. –≠—Ç–æ—Ç –∏–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –∫–∞–∫ –¥–ª—è –∏–Ω–¥–∏–≤–∏–¥—É–∞–ª—å–Ω–æ–π, —Ç–∞–∫ –∏ –¥–ª—è –∫–æ–º–∞–Ω–¥–Ω–æ–π —Ä–∞–±–æ—Ç—ã.
Git –ø–æ–∑–≤–æ–ª—è–µ—Ç —Å–æ—Ö—Ä–∞–Ω—è—Ç—å –∏–∑–º–µ–Ω–µ–Ω–∏—è –ª–æ–∫–∞–ª—å–Ω–æ –∏ –ø—Ä–∏ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ—Å—Ç–∏ –≤–æ–∑–≤—Ä–∞—â–∞—Ç—å—Å—è –∫ –ø—Ä–µ–¥—ã–¥—É—â–∏–º –≤–µ—Ä—Å–∏—è–º –ø—Ä–æ–µ–∫—Ç–∞. –¢–∞–∫–∂–µ –º–æ–∂–Ω–æ —Å–æ–∑–¥–∞—Ç—å —É–¥–∞–ª—ë–Ω–Ω—É—é –∫–æ–ø–∏—é –Ω–∞ —Ö–æ—Å—Ç–∏–Ω–≥-–ø–ª–∞—Ç—Ñ–æ—Ä–º–µ, –∫–æ—Ç–æ—Ä–∞—è —Ä–∞–±–æ—Ç–∞–µ—Ç —Å Git, –∏ –ø–æ–¥–µ–ª–∏—Ç—å—Å—è —Ä–µ–∑—É–ª—å—Ç–∞—Ç–æ–º —Å –¥—Ä—É–≥–∏–º–∏.

### –û—Å–Ω–æ–≤–Ω—ã–µ —ç—Ç–∞–ø—ã 


1. –°–æ–∑–¥–∞–µ–º –ø–∞–ø–∫—É, –≤ –∫–æ—Ç–æ—Ä–æ–π –±—É–¥–µ—Ç –Ω–∞—Ö–æ–¥–∏—Ç—Å—è –Ω–∞—à –ø—Ä–æ–µ–∫—Ç


2. –ß—Ç–æ–±—ã Git –Ω–∞—á–∞–ª –æ—Ç—Å–ª–µ–∂–∏–≤–∞—Ç—å –∏–∑–º–µ–Ω–µ–Ω–∏—è –≤ –ø—Ä–æ–µ–∫—Ç–µ, –ø–∞–ø–∫—É —Å —Ñ–∞–π–ª–∞–º–∏ —ç—Ç–æ–≥–æ –ø—Ä–æ–µ–∫—Ç–∞ –Ω—É–∂–Ω–æ —Å–¥–µ–ª–∞—Ç—å Git-—Ä–µ–ø–æ–∑–∏—Ç–æ—Ä–∏–µ–º (–æ—Ç –∞–Ω–≥–ª. repository ‚Äî ¬´—Ö—Ä–∞–Ω–∏–ª–∏—â–µ¬ª). –î–ª—è —ç—Ç–æ–≥–æ —Å–ª–µ–¥—É–µ—Ç –ø–µ—Ä–µ–º–µ—Å—Ç–∏—Ç—å—Å—è –≤¬†–Ω–µ—ë –∏ –≤–≤–µ—Å—Ç–∏ –∫–æ–º–∞–Ω–¥—É ```git init```.


2.1 –ï—Å–ª–∏ –≤—ã —Å–ª—É—á–∞–π–Ω–æ —Å–¥–µ–ª–∞–ª–∏ Git-—Ä–µ–ø–æ–∑–∏—Ç–æ—Ä–∏–µ–º –Ω–µ —Ç—É –ø–∞–ø–∫—É, –µ—ë –º–æ–∂–Ω–æ ¬´—Ä–∞–∑–≥–∏—Ç–∏—Ç—å¬ª. –î–ª—è —ç—Ç–æ–≥–æ –Ω—É–∂–Ω–æ —É–¥–∞–ª–∏—Ç—å —Å–∫—Ä—ã—Ç—É—é –ø–æ–¥–ø–∞–ø–∫—É `.git`.

3. –ü–æ—Å–ª–µ –∏–Ω–∏—Ü–∏–∞–ª–∏–∑–∞—Ü–∏–∏ —Ä–µ–ø–æ–∑–∏—Ç–æ—Ä–∏—è –∑–∞–ø—É—Å—Ç–∏—Ç–µ –∫–æ–º–∞–Ω–¥—É ```git status``` (–æ—Ç –∞–Ω–≥–ª. status ‚Äî ¬´—Å—Ç–∞—Ç—É—Å¬ª, ¬´—Å–æ—Å—Ç–æ—è–Ω–∏–µ¬ª) ‚Äî –æ–Ω–∞ –ø–æ–∫–∞–∑—ã–≤–∞–µ—Ç —Ç–µ–∫—É—â–µ–µ —Å–æ—Å—Ç–æ—è–Ω–∏–µ —Ä–µ–ø–æ–∑–∏—Ç–æ—Ä–∏—è.


4. –ü–æ—Å–ª–µ –ø–æ–¥–≥–æ—Ç–∞–≤–ª–∏–≤–∞–µ–º —Ñ–∞–π–ª –∫ —Å–æ—Ö—Ä–∞–Ω–µ–Ω–∏—é —Å –ø–æ–º–æ—â–±—é –∫–æ–º–∞–Ω–¥—ã ``` git add --all | git add –Ω–∞–∑–≤–∞–Ω–∏–µ_—Ñ–∞–π–ª–∞ | git add . (–¥–æ–±–∞–≤–∏—Ç—å –≤—Å—é —Ç–µ–∫—É—â—É—é –ø–∞–ø–∫—É)```


5. –°–æ–∑–¥–∞–µ–º commit, –∫–æ—Ç–æ—Ä—ã–π –≥–∞—Ä–∞–Ω—Ç–∏—Ä—É–µ—Ç, —á—Ç–æ –∏–∑–º–µ–Ω–µ–Ω–∏—è –±—É–¥—É—Ç —Å–æ—Ö—Ä–∞–Ω–µ–Ω—ã –≤ –∏—Å—Ç–æ—Ä–∏–∏ –∏ –ø—Ä–∏ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ—Å—Ç–∏ –∫ –Ω–∏–º –º–æ–∂–Ω–æ –±—É–¥–µ—Ç ¬´–æ—Ç–∫–∞—Ç–∏—Ç—å—Å—è¬ª, —Å –ø–æ–º–æ—â—å—é –∫–æ–º–∞–Ω–¥—ã ```git commit -m '–°–æ–æ–±—â–µ–Ω–∏–µ –æ –∫–æ–º–º–∏—Ç–µ'```

6. –ß—Ç–æ–±—ã —É–≤–∏–¥–µ—Ç—å –≤—Å–µ –∫–æ–º–º–∏—Ç—ã, –≤–≤–µ–¥–∏—Ç–µ –∫–æ–º–∞–Ω–¥—É ```git log``` (–æ—Ç –∞–Ω–≥–ª. *log* ‚Äî ¬´–∂—É—Ä–Ω–∞–ª [–∑–∞–ø–∏—Å–µ–π]¬ª).

–û–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ, —á—Ç–æ –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é ```git log``` –≤—ã–≤–æ–¥–∏—Ç –∫–æ–º–º–∏—Ç—ã –≤ –æ–±—Ä–∞—Ç–Ω–æ–º —Ö—Ä–æ–Ω–æ–ª–æ–≥–∏—á–µ—Å–∫–æ–º –ø–æ—Ä—è–¥–∫–µ ‚Äî –ø–æ—Å–ª–µ–¥–Ω–∏–µ –∫–æ–º–º–∏—Ç—ã –æ–∫–∞–∑—ã–≤–∞—é—Ç—Å—è –ø–µ—Ä–≤—ã–º–∏ —Å–≤–µ—Ä—Ö—É. –í —ç—Ç–æ–º –º–æ–∂–Ω–æ —É–±–µ–¥–∏—Ç—å—Å—è, –µ—Å–ª–∏ –ø–æ—Å–º–æ—Ç—Ä–µ—Ç—å –Ω–∞ –¥–∞—Ç—É –∏ –≤—Ä–µ–º—è –∏—Ö —Å–æ–∑–¥–∞–Ω–∏—è.

7. –°–≤—è–∑–∞—Ç—å –ª–æ–∫–∞–ª—å–Ω—ã–π –∏ —É–¥–∞–ª–µ–Ω–Ω—ã–π —Ä–µ–ø–æ–∑–∏—Ç–æ—Ä–∏–π 

### –ü—Ä–∏–≤—è–∑–∞—Ç—å —É–¥–∞–ª—ë–Ω–Ω—ã–π —Ä–µ–ø–æ–∑–∏—Ç–æ—Ä–∏–π –∫ –ª–æ–∫–∞–ª—å–Ω–æ–º—É ‚Äî¬†`git remote add`

	
7.1.1 –ü–µ—Ä–µ–π–¥–∏—Ç–µ –Ω–∞ —Å—Ç—Ä–∞–Ω–∏—Ü—É —É–¥–∞–ª—ë–Ω–Ω–æ–≥–æ —Ä–µ–ø–æ–∑–∏—Ç–æ—Ä–∏—è, –≤—ã–±–µ—Ä–∏—Ç–µ —Ç–∏–ø SSH –∏ —Å–∫–æ–ø–∏—Ä—É–π—Ç–µ URL. –ö–Ω–æ–ø–∫–∞ —Å–ø—Ä–∞–≤–∞ –ø–æ–∑–≤–æ–ª–∏—Ç —Å–¥–µ–ª–∞—Ç—å —ç—Ç–æ –º–≥–Ω–æ–≤–µ–Ω–Ω–æ.



7.1.2 –û—Ç–∫—Ä–æ–π—Ç–µ –∫–æ–Ω—Å–æ–ª—å, –ø–µ—Ä–µ–π–¥–∏—Ç–µ –≤ –∫–∞—Ç–∞–ª–æ–≥ –ª–æ–∫–∞–ª—å–Ω–æ–≥–æ —Ä–µ–ø–æ–∑–∏—Ç–æ—Ä–∏—è –∏ –≤–≤–µ–¥–∏—Ç–µ –∫–æ–º–∞–Ω–¥—É `git remote add` (–æ—Ç –∞–Ω–≥–ª. *remote* ‚Äî ¬´—É–¥–∞–ª—ë–Ω–Ω—ã–π¬ª –∏ *add* ‚Äî ¬´–¥–æ–±–∞–≤–∏—Ç—å¬ª).


–ö–æ–º–∞–Ω–¥–µ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –ø–µ—Ä–µ–¥–∞—Ç—å –¥–≤–∞ –ø–∞—Ä–∞–º–µ—Ç—Ä–∞: –∏–º—è —É–¥–∞–ª—ë–Ω–Ω–æ–≥–æ —Ä–µ–ø–æ–∑–∏—Ç–æ—Ä–∏—è –∏¬†–µ–≥–æ URL. –í –∫–∞—á–µ—Å—Ç–≤–µ –∏–º–µ–Ω–∏ –∏—Å–ø–æ–ª—å–∑—É–π—Ç–µ —Å–ª–æ–≤–æ `origin`. –ê URL –≤—ã —Å–∫–æ–ø–∏—Ä–æ–≤–∞–ª–∏ —Å–æ —Å—Ç—Ä–∞–Ω–∏—Ü—ã —É–¥–∞–ª—ë–Ω–Ω–æ–≥–æ —Ä–µ–ø–æ–∑–∏—Ç–æ—Ä–∏—è.


–£–±–µ–¥–∏—Ç—å—Å—è, —á—Ç–æ —Ä–µ–ø–æ–∑–∏—Ç–æ—Ä–∏–∏ —Å–≤—è–∑–∞–Ω—ã, ‚Äî¬†git remote -v

